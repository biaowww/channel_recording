它解决什么

🎯

只录那一个程序

指定哪个程序,就只录它的声音。你的微信提示音、音乐、键盘声都不会混进去 —— 不是"录整个电脑"。

🖼️

有音,也有画

自动识别投屏翻页,每一页存成图片,收工时合成一个 PDF / Word。听回放时对得上当时在讲哪一页。

🤖

无人值守

会议软件关掉、或安静超过设定时间,它自己停、自己把音频压成小文件。你可以放着去干别的。

🎤

可以带上你自己

勾一下就把麦克风混进同一个文件 —— 对方说的和你说的,都在一条音轨里。

🗜️

文件不会大到没法用

自动转成 AAC,约 0.7MB/分钟。两小时的会大概 80MB,微信能发、网盘能传。

🪟

想录哪块自己划

整块屏幕、某个窗口(会跟着窗口走)、或者鼠标拖一个框 —— 只要 PPT 那一块也行。

三步开录

  1. 选要录的程序 —— 列表里挑,正在出声的会标 🔊
  2. 勾要不要抓画面 —— 要的话再选录屏幕的哪一块
  3. 点开始,然后就不用管了 —— 结束后音频和 PDF 都在你指定的文件夹里

都能拿来录什么

线上会议回看 网课 / 培训 播客 网页里的音视频 直播 / 分享会 带投屏的任何讲解

为什么它录得到

它录的是声音播放到系统之后的那一路 —— 也就是你耳朵听到的那份, 而不是去调用某个软件自带的录制功能。所以它不依赖对方软件给不给"录制权限"。 画面走的是 Windows 现代捕获接口,硬件加速的内容也不会花屏,窗口被挡住也照抓。

常见问题

要装什么吗?

不用。下载下来是一个 exe,双击就开。运行环境已经打包在里面了(所以文件有点大,约 176MB)。

只能 Windows?

是的。它用的是 Windows 才有的进程级音频捕获(需要 Win10 2004 / build 19041 以上,Win11 直接可用)。

录出来的文件放哪?

界面上「存放位置」那一行随时看得到,可以改成你想要的任意文件夹,改完会记住。旁边有按钮一键打开。

录出来没声音怎么办?

八成是「录制目标」选错了程序。录制时界面有实时音量条,一直是空的就说明选错了,当场就能发现,不用等录完。

收费吗?会不会传我的东西?

完全免费、开源(MIT)。全程只在你自己电脑上跑,不联网、不上传任何内容。源码在 GitHub 上,可以自己看。
